# [연결 요소의 개수](https://github.com/malvr00/Java-algorithm/blob/master/backjoon/dfs_bfs/step11/src/Main.java)
방향 없는 그래프가 주어졌을 때, 연결 요소 (Connected Component)의 개수를 구하는 프로그램을 작성하시오.

([자세한 문제는 백준 참고](https://www.acmicpc.net/problem/11724)) <br/>

## 해법
* 맨 처음 입력 값들을 양방향 배열에 저장한다.
  * 배열의 `INDEX`가 `node`의 개수를 나타낸다.
* 배열의 크기만큼 반복문을 돌면서 `DFS`탐색을 진행한다.
  * 탐색을 진행하면서 처음 방문한 위치는 `TRUE`로 초기화한 후 카운트를 1 증가시켜 연결된 노드 개수를 카운팅한다. 그 후 재귀호출한다.
  * 탐색을 진행하면서 방문한 위치일 경우 해당 탐색은 종료하고 다음 탐색을 진행한다.
* 마지막으로 연결된 노드의 개수가 `0 이상` 일 경우 노드가 연결된 것으로 판단하여 연결 요소 개수를 증가시킨다.

## 핵심
* 탐색을 `1 ~ N`까지 `DFS`탐색을 진행하면 쉽게 문제를 풀 수 있다.
