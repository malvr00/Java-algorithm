# [단지번호붙이기](https://github.com/malvr00/Java-algorithm/blob/master/backjoon/dfs_bfs/step4/src/Main.java)
<그림 1>과 같이 정사각형 모양의 지도가 있다. 1은 집이 있는 곳을, 0은 집이 없는 곳을 나타낸다. 철수는 이 지도를 가지고 연결된 집의 모임인 단지를 정의하고, 단지에 번호를 붙이려 한다. 여기서 연결되었다는 것은 어떤 집이 좌우, 혹은 아래위로 다른 집이 있는 경우를 말한다. 대각선상에 집이 있는 경우는 연결된 것이 아니다. <그림 2>는 <그림 1>을 단지별로 번호를 붙인 것이다. 지도를 입력하여 단지수를 출력하고, 각 단지에 속하는 집의 수를 오름차순으로 정렬하여 출력하는 프로그램을 작성하시오.

([자세한 문제는 백준 참고](https://www.acmicpc.net/problem/2667)) <br/>

## 해법
* `board` 정사각형 크기만큼 반복문을 돌면서 탐색을 진행해야한다.
* 탐색에서 중요한점은 중복탐색을 방지해야한다.
  * `board` 크기만큼 체크배열을 하나 만들어서 방문한곳은 `TRUE`로 표기한 후 재방문을 방지한다.
* 탐색을 진행하면서 재방문했는지 확인과 해당 위치가 집이있는 위치(`1`)인지 확인 후 두 조건이 충족된다면 `카운트 1 증가`와 체크배열에 해당 위치를 `TRUE`로 방문했다는 표시를 한다.
* 탐색이 완료됐을 때 카운트 값이 `0`보다 크다면 해당 시작지점 좌표에서 집이 존재한다는 의미이므로 `List`에 카운트 수를 저장하고, 단지 수를 세는 변수 `cnt`에 `+ 1` 해준다.
* 마지막으로 문제에서 단지의 집 개수를 오름차순으로 내보내달라고 했기 때문에 `List`를 오름차순으로 정렬 후 출력한다.

## 핵심
* 중복방문만 막으면 쉽게 풀 수 있다.

## 참고
* 해당 문제를 풀 때 따로 체크배열을 만들어서 해결했는데, 집 위치를 가지고 있는 `board` 배열을 활용해서 재방문을 방지할 수 있다.
  * 집의 위치의 값은 무조건 `1`이기 때문에 이를 잘 활용하면 된다.
