# [동전 1](https://github.com/malvr00/Java-algorithm/blob/master/backjoon/dp/step15/src/Main.java)
n가지 종류의 동전이 있다. 각각의 동전이 나타내는 가치는 다르다. 이 동전을 적당히 사용해서, 그 가치의 합이 k원이 되도록 하고 싶다. 그 경우의 수를 구하시오. 각각의 동전은 몇 개라도 사용할 수 있다.<br/>
<br/>
사용한 동전의 구성이 같은데, 순서만 다른 것은 같은 경우이다.
([자세한 문제는 백준 참고](https://www.acmicpc.net/problem/2293)) <br/>

## 해법
* 먼저 구해야하는 `금액 + 1` 크기만큼 배열 `dp`를 만든다.
* `dp[0]`의 값을 `1`로 초기화한다.
  * `1`로 초기화하는 이유는 `j` 금액 크기에 경우의 수를 1 씩 누적합을 진행하기 위해 `[0]`을 1로 초기화한다.
    * `dp[0]`을 `1`로 초기화하지 않고 `0`으로 초기화 후 `+ 1` 씩 연산을 더해줄 경우 `1 원` 짜리 동전으로 `2 원`을 구하는 경우의 수가 `2`가 나오는 문제가 발생한다.
* 동전의 종류 만큼 반복문을 돌면서 동전의 금액을 기준으로 구해야하는 최종 금액까지 추가로 반복문을 돌면서 누적합을 진행한다.
  * 점화식은 `dp[j] += dp[j - coin]`이다.

## 핵심
* 맨 처음 `0`번 배열 초기화 값 선정이 중요하다.
