# [선물 전달](https://github.com/malvr00/Java-algorithm/blob/master/backjoon/dp/step32/src/Main.java)
이번 ACM-ICPC 대회에 참가한 모든 사람들은 선물을 하나씩 준비했다.<br/>
<br/>
대회가 끝나고 난 후에 각자 선물을 전달하려고 할 때, 선물을 나누는 경우의 수를 구하는 프로그램을 작성하시오.<br/>
<br/>
모든 사람은 선물을 하나씩 받으며, 자기의 선물을 자기가 받는 경우는 없다.

([자세한 문제는 백준 참고](https://www.acmicpc.net/problem/1947)) <br/>

## 해법
* 배열을 `dp[N + 1]` 생성한다.
* `N` 까지 반복문을 돌면서 점화식을 실행한다.
* 점화식은 `dp[i] = i - 1 * (dp[i - 2] + dp[i - 1])` 이다.
  * `i - 1`을 곱하는 이유는 문제에서 자기 자신의 선물은 받을 수 없기 때문에 `i`가 선물 받을 수 있는 경우는 `i - 1`이기 때문이다.

## 핵심
* 인덱스의 정의가 중요한데, `DP` 알고리즘을 풀 때 경우의 수를 전이 시키기 위해서는 대부분 `dp[i - 2] + dp[i - 1]` 점화식을 사용한다. 어떤 문제든 파악해서 미리 알고 있으면 좋다.
