# [파도반 수열](https://github.com/malvr00/Java-algorithm/blob/master/backjoon/dp/step16/src/Main.java)
오른쪽 그림과 같이 삼각형이 나선 모양으로 놓여져 있다. 첫 삼각형은 정삼각형으로 변의 길이는 1이다. 그 다음에는 다음과 같은 과정으로 정삼각형을 계속 추가한다. 나선에서 가장 긴 변의 길이를 k라 했을 때, 그 변에 길이가 k인 정삼각형을 추가한다.<br/>
<br/>
파도반 수열 P(N)은 나선에 있는 정삼각형의 변의 길이이다. P(1)부터 P(10)까지 첫 10개 숫자는 1, 1, 1, 2, 2, 3, 4, 5, 7, 9이다.<br/>
<br/>
N이 주어졌을 때, P(N)을 구하는 프로그램을 작성하시오.

([자세한 문제는 백준 참고](https://www.acmicpc.net/problem/9461)) <br/>

## 해법
* `N + 1` 크기 만큼 배열을 생성해준다.
* 지문에서 `P(N)`을 `10`까지 정의해줬는데 `1`을 가지는 `P(3)`까지는 미리 `1`로 초기화 해둔다.
  * 여기서 주의할 점이 `N` 의 값이 `3 이하` 일 경우 `ArrayIndexOutOfBounds` 에러가 발생할 수 있기 때문에 `N`이 `4 미만`일 경우 `1`을 반환하도록 한다.
* `N`이 `4 이상`일 경우 반복문을 돌면서 배열에 값을 쌓아준다.
  * 점화식은 `dp[i] = dp[i - 3] + dp[i - 2]` 이다. 

## 핵심
* 문제 지문을 잘 읽고 파악하는게 중요하다.
