# [상자넣기](https://github.com/malvr00/Java-algorithm/blob/master/backjoon/dp/step21/src/Main.java)
정육면체 모양의 상자가 일렬로 늘어서 있다. 상자마다 크기가 주어져 있는데, 앞에 있는 상자의 크기가 뒤에 있는 상자의 크기보다 작으면, 앞에 있는 상자를 뒤에 있는 상자 안에 넣을 수가 있다. 예를 들어 앞에서부터 순서대로 크기가 (1, 5, 2, 3, 7)인 5개의 상자가 있다면, 크기 1인 상자를 크기 5인 상자에 넣고, 다시 이 상자를 크기 7인 상자 안에 넣을 수 있다. 하지만 이렇게 상자를 넣을 수 있는 방법은 여러 가지가 있을 수 있다. 앞의 예에서 차례대로 크기가 1, 2, 3, 7인 상자를 선택하면 총 4개의 상자가 한 개의 상자에 들어가게 된다.<br/>
<br/>
상자의 크기가 주어질 때, 한 번에 넣을 수 있는 최대의 상자 개수를 출력하는 프로그램을 작성하시오.

([자세한 문제는 백준 참고](https://www.acmicpc.net/problem/1965)) <br/>

## 해법
* 상자 개수 `N + 1` 크기만큼의 배열을 만든다.
* `LIS`방식으로 문제를 해결하면 되는데 인덱스 정의를 잘 해야한다.
  * `인덱스 = n 까지 넣은 총 상자 수`
* `N` 크기 만큼 반복문을 하면서 현재 상자보다 이전 상자들이 더 작다면 카운트를 늘려야한다.
* 점화식은 다음과 같다. `if(boxes[j] < boxes[i]) dp[i] = max(dp[i], dp[j] + 1)`
  * 핵심은 이전 상자가 더 작을때만 값을 비교해 더 큰 값을 넣어야한다.
* 마지막으로 `n` 위치중에 가장 큰 상자 개수를 반환하면된다. 

## 핵심
* 배열 인덱스의 정의가 중요하다.
  * `[n] -> n 까지 넣은 최대 상자 개수`
* 자기자신도 넣을 수 있는 최대 상자 개수에 포함된다.
