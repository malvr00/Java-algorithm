# [줄세우기](https://github.com/malvr00/Java-algorithm/blob/master/backjoon/dp/step24/src/Main.java)
KOI 어린이집에는 N명의 아이들이 있다. 오늘은 소풍을 가는 날이다. 선생님은 1번부터 N번까지 번호가 적혀있는 번호표를 아이들의 가슴에 붙여주었다. 선생님은 아이들을 효과적으로 보호하기 위해 목적지까지 번호순서대로 일렬로 서서 걸어가도록 하였다. 이동 도중에 보니 아이들의 번호순서가 바뀌었다. 그래서 선생님은 다시 번호 순서대로 줄을 세우기 위해서 아이들의 위치를 옮기려고 한다. 그리고 아이들이 혼란스러워하지 않도록 하기 위해 위치를 옮기는 아이들의 수를 최소로 하려고 한다.<br/>
<br/>
예를 들어, 7명의 아이들이 다음과 같은 순서대로 줄을 서 있다고 하자.<br/>
<br/>
3 7 5 2 6 1 4<br/>
<br/>
아이들을 순서대로 줄을 세우기 위해, 먼저 4번 아이를 7번 아이의 뒤로 옮겨보자. 그러면 다음과 같은 순서가 된다.<br/>
<br/>
3 7 4 5 2 6 1<br/>
<br/>
이제, 7번 아이를 맨 뒤로 옮긴다.<br/>
<br/>
3 4 5 2 6 1 7<br/>
<br/>
다음 1번 아이를 맨 앞으로 옮긴다.<br/>
<br/>
1 3 4 5 2 6 7<br/>
<br/>
마지막으로 2번 아이를 1번 아이의 뒤로 옮기면 번호 순서대로 배치된다.<br/>
<br/>
1 2 3 4 5 6 7<br/>
<br/>
위의 방법으로 모두 4명의 아이를 옮겨 번호 순서대로 줄을 세운다. 위의 예에서 3명의 아이만을 옮겨서는 순서대로 배치할 수가 없다. 따라서, 4명을 옮기는 것이 가장 적은 수의 아이를 옮기는 것이다.<br/>
<br/>
N명의 아이들이 임의의 순서로 줄을 서 있을 때, 번호 순서대로 배치하기 위해 옮겨지는 아이의 최소 수를 구하는 프로그램을 작성하시오.<br/>

([자세한 문제는 백준 참고](https://www.acmicpc.net/problem/2631)) <br/>

## 해법
* 먼저 아이들의 수 `N` 만큼 배열을 만든다.
* 이 문제는 최장 증가 부분수열을 만족하는 최대 개수를 찾아야하기 때문에 `lis` 변수를 하나 만든다.
  * 아이들을 줄 세울 때 `i` 위치의 아이와 `j` 위치의 아이가 서로 증가 부분수열의 조건을 만족한다면 해당 `i`와 `j`의 아이는 위치 이동을하지 않아도 되기 때문에 이동을 하지 않아도 되는 아이들을 찾아야한다.
  * 결과적으로 증가 부분수열을 만족하는 아이들의 최대 수를 찾으면 그 수에 `N`을 빼주면 이동해야하는 아이만 찾을 수 있고 그게 정답이된다.
* `N` 만큼 반복문을 돌면서 아래와 같은 점화식을 실행한다.
  * `for(j = 0; j < i; j++) if(children[j] < children[i]) dp[i] = max(dp[i], dp[j + 1])`
* 마지막으로 `N - lis`를 반환한다.

## 핵심
* 지문에서 번호가 `+ 1` 씩 증가하는 증가 부분수열의 문제이기 때문에 `LIS` 방법을 활용해서 문제를 해결하면된다.
  * 인덱스의 정의는 `i` 번 줄까지 이동하지 않아도 되는 아이의 수를 저장하면된다.

## 문제점
* 아직 `LIS` 문제가 약해서 문제 푸는데 조금 오래걸리는 것 같다.
